Extensions
==============

MEmilio provides several extensions to interact with external data structures or software. 

- :doc:`SBML <sbml>` can be used to semi-automatically create models in MEmilio given a model description in the SBML format.
- :doc:`Epidata <../python/memilio_epidata>` provides a Python interface to access epidemiological data.
