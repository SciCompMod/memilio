# ODE-based SEIR model

This directory contains a metapopulation model implementation based on an ODE formulation, modeling the mobility implicitly by including the impact of other regions into the force of infection.
To get started, check out the [official documentation](https://memilio.readthedocs.io/en/latest/cpp/ode_metapop.html) 
or the code example:

- [examples/ode_seir_metapop.cpp](../../examples/ode_seir_metapop.cpp)

For the metapopulation model approach that models the mobility explicitly, have a look at the [corresponding documentation](https://memilio.readthedocs.io/en/latest/cpp/graph_metapop.html).
