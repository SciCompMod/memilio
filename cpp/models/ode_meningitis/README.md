# ODE-based Low-and-high-susceptible-SCIR-type model

This directory contains a model implementation based on an ODE formulation
of the meningitis model as published in https://doi.org/10.1155/2022/9084283. 
To get started, check out the [official documentation](https://memilio.readthedocs.io/en/latest/cpp/models/omeng.html)
